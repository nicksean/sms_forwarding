蜂窝数据设置

注意：开启会导致收不到短信，模组会掉为纯数据附着，丢失承载短信的 CS/SGs 信令域（漫游时尤其明显）。本机经 WiFi 转发，无需开流量，保持关闭即可。

默认关闭=禁用 PDP，零流量。仅在需要保号 HTTP payload / 蜂窝 IP 时临时开启。

同手机“数据漫游”：关闭后设备处于漫游网络(CEREG=5)时不激活蜂窝数据、不跑漫游流量，短信照常接收(走 CS/IMS)。境外/漫游 eSIM 需保持开启，否则可能没有数据、部分卡收不到短信。

从列表选运营商或手动填 PLMN(MCC+MNC，国外同理)。仅能选 SIM 可接入的网络，锁定不可达会失网；不确定留空自动。

多数 eSIM/数据卡不通过 CNUM 上报本机号码，模组读不到时概览会显示这里的手填值。换卡后请更新或清空，否则会一直显示旧号码。
